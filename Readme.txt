Before running our program, please add all the library JAR files, which locates in the directory FeatureBased_PPIExtraction_ICTAI2017\Libraries to the Java Build Path in Eclipse. 

PPIextraction.java is used for extraction of PPIs from the text (corpus) which locates in FeatureBased_PPIExtraction_ICTAI2017\XMLFiles.
evalCrossValidation.java is used for cross-validation evaluation.
evalCrossCorpus.java used for cross-corpus evaluation.
PPIextraction.java, evalCrossValidation.java, and evalCrossCorpus.java are independent.
